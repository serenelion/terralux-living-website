# TerraLux Living - Lifestyle Membership

A $25,000/year global lifestyle membership for conscious creators.

## Overview
TerraLux Living connects members to healing homes, biohacking centers, regenerative villages, and a curated benefits network worldwide.

## Features
- **Four Pillars**: Home, Tech, Living, Health
- **Global Network**: Villages and wellness clubs in 15+ locations
- **Benefits Network**: 50% off masterminds, biohacking access, mobility programs
- **Partner Ecosystem**: 9+ curated partners (Integrative Organizing, Midtown Biohack, Oya New Earth, Executive Healing, Inokim, German Kitchen Center, Spatial Network, XZEN TV)
- **Founding Member Status**: Permanent priority access

## Tech Stack
- Pure HTML/CSS/JS (no build step)
- Netlify Forms for applications
- Indigo + Gold brand system
- Responsive design

## Local Development

```bash
# Serve locally
python3 -m http.server 8000
# Or
npx serve .
```

Visit: `http://localhost:8000`
